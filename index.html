RESPONSIVE DESIGN & WEB DEVELOPMENT

DESCRIPTION

This module will provide the learner with the skills required to design and develop interactive & responsive systems.

LEARNING OUTCOMES:

On completion of this module the learner will/should be able to:

  • Identify and explain the theories and practices associated with the design of an interactive system.
  • Prototype and implement an interactive and responsive system.
  • Publish and maintain an interactive system.
  • Evaluate and document the usability of an interactive system.

INDICATIVE SYLLABUS:

Design Principles & Concepts

  • Usability, user experience, accessibility
  • Prototyping - concepts, techniques, wireframes
  • Cognitive aspects of design e.g. readability, information perception
  • Interface types e.g. GUI, multimedia, web, mobile

Responsive & Interactive Systems Development

  • Responsive design requirement & benefits
  • Responsive design theory & techniques e.g. CSS media queries, grid based layoutsetc.
  • Techniques for adding interactivity e.g. JavaScript, DHTML, libraries
  • Code management tools & editors e.g. GitHub, Brackets

Publishing and Maintaining Interactive Systems

  • Choosing a host, managing resources, deployment options, registration, optimisation

Interactive Systems Evaluation

  • Evaluation frameworks, usability evaluation
  • Inspections - Heuristic evaluation & walkthroughs, accessibility

FULL TIME MODE WORKLOAD(PER WEEK):

  • Lab Lectures:   2 hours
  • Practicals:         2 hours

TEACHING AND LEARNING STRATEGIES

This module aims to use active learning by combining theoretical aspects of responsive & interactive design with practical experience of developing systems for various computing platforms such as the web. This will give the student a strong overall understanding of the subject.

RECOMMENDED ASSESSMENT TECHNIQUES:

  • Weekly Practicals                     10%
  • Performance Evaluation         30%
  • Project                                       60%

Typical Project Assignment

  • Create a responsive site to market your local town using CSS Grid and Media queries applying the principles of good design
  • Design, develop and publish a responsive website for a club or society using HTML, CSS and JavaScript, using CSS3 Media Queries and CSS Grid

REPEAT ASSESSMENTS:

Learners who achieve a combined grade of less than 40% in the module will need to complete a substantial individual web project.
